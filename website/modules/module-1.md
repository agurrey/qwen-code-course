# Module 1: What Is Qwen Code?

> **Prerequisites:** Module 0
> **Time:** ~15 minutes
> **Lessons:** 4

## What You'll Learn

- What Qwen Code actually is (and isn't)
- How it "thinks" in tools, not just text
- The safety model: approval modes and why they matter

## Why This Module Matters

Without these mental models, every other module will feel confusing. This is the "understand the tool before you use it" module. It's still hands-on — you'll test the boundaries of what Qwen Code knows and doesn't know.

## Lessons

### Lesson 1.1: What Qwen Code Actually Is
The core mental model: Qwen Code is a large language model with tools. It only knows what it reads.

### Lesson 1.2: What Qwen Code Is NOT
The misconceptions that cause the most damage: it's not a chatbot, not a search engine, not omniscient.

### Lesson 1.3: How Qwen Code Thinks in Tools
The internal workflow: request → tool selection → execution → result → next tool.

### Lesson 1.4: Safety and Approval Modes
The four approval modes and when to use each one.

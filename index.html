Why This Approach?

Learn by Doing

Every lesson has exercises with real output. No "read this and understand" — you always DO something.

No Context Switching

The course IS the tool you're learning. No videos, no separate apps. Everything happens in your terminal.

Self-Paced

Start, stop, come back. Your progress is tracked automatically. ~6 hours total, at your own speed.

Open Source

Built by the community, for the community. Fork it, contribute, make it better. MIT licensed.

Quick Start

# Install Qwen Code
npm install -g @qwen-code/qwen-code

# Clone and install the course
git clone https://github.com/agurrey/qwen-code-course.git
cd qwen-code-course
bash scripts/install.sh

# Start learning
qwen
# Then type: /course start

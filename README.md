# Qwen Code Course

> **Learn Qwen Code by using Qwen Code.** No downloads, no videos, no separate apps. The entire course runs inside your terminal.

## Who This Is For

**Total beginners.** If you've never opened a terminal, this course starts at zero. We explain what a terminal is, why it matters, and how to not be scared of a blinking cursor.

## How It Works

1. Install Qwen Code
2. Install this course (copy commands + lessons into your `~/.qwen/` directory)
3. Type `/course start`
4. Work through interactive lessons — each one has exercises you **actually do**, not just read
5. Track your progress automatically
6. Generate a personalized cheat sheet when you're done

## Quick Start

```bash
# Clone this repo
git clone https://github.com/agurrey/qwen-code-course.git
cd qwen-code-course

# Install the course into your Qwen Code setup
./scripts/install.sh

# Start learning
qwen
# Then type: /course start
```

## Why This Approach?

Most AI tool courses are videos you watch or docs you read. This one is **the tool teaching itself**. Every lesson runs inside Qwen Code, with exercises that produce real output. You learn by doing, not by watching.

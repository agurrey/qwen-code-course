# Learn Qwen Code. By Using Qwen Code.

> A free, interactive course for total beginners. No prior terminal experience needed.

## What Is This?

Most courses teach you to watch or read. This one teaches you to **do**. Every lesson runs inside Qwen Code itself — an AI coding assistant that lives in your terminal. You learn by giving it tasks, watching it work, and building things yourself.

## Who Is This For?

**Total beginners.** If you've never opened a terminal, this course starts at zero. We explain what a terminal is, why it matters, and how to not be scared of a blinking cursor.

If you've used a terminal before, you can skip the early modules. If you've used AI coding tools, you can skip even further.

## Why This Approach?

- **Learn by doing, not watching** — Every lesson has exercises with real output
- **No context switching** — The course IS the tool you're learning
- **Self-paced** — Start, stop, come back. Your progress is tracked
- **Personalized** — The course adapts to your experience level and goals
- **Open source** — Built by the community, for the community

## Quick Start

```bash
# Install Qwen Code
npm install -g @qwen-code/qwen-code

# Install the course
git clone https://github.com/agurrey/qwen-code-course.git
cd qwen-code-course
bash scripts/install.sh

# Start learning
qwen
# Then type: /course start
```

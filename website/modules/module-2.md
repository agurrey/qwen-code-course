# Module 2: Your First Commands

> **Prerequisites:** Module 0, Module 1
> **Time:** ~30 minutes
> **Lessons:** 6

## What You'll Learn

The six core capabilities that cover 80% of daily Qwen Code usage:

1. **Read** — Read and understand files
2. **Edit** — Modify files through natural language
3. **Shell** — Run terminal commands safely
4. **Grep** — Search file contents for patterns
5. **Glob** — Find files by name pattern
6. **Web Fetch** — Retrieve and extract web content

## Why This Module Is the Most Important

These six capabilities are what you'll use every single day. The later modules (MCP, agents, hooks) build on these. If you master Module 2, you can already be productive with Qwen Code.

## Lessons

### Lesson 2.1: Reading Files
Read, summarize, analyze, and transform file content.

### Lesson 2.2: Editing Files
Modify files without opening a text editor.

### Lesson 2.3: Running Shell Commands
Execute terminal commands through Qwen Code with safety checks.

### Lesson 2.4: Searching with Grep
Find text patterns across your entire codebase.

### Lesson 2.5: Finding Files with Glob
Locate files by name pattern across directory trees.

### Lesson 2.6: Fetching Web Content
Retrieve and extract information from websites.

## Combined Exercise

Lesson 2.6 includes a capstone exercise that combines all six capabilities into one project.

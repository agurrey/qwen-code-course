# Module 0: Your First 5 Minutes

> **Prerequisites:** None. This is where everyone starts.
> **Time:** ~5 minutes
> **Lessons:** 3

## What You'll Learn

- What a terminal is and why it matters
- How to install Qwen Code
- How to give Qwen Code its first task and see a real result

## Why This Module Exists

Most courses spend 30 minutes explaining what an AI agent is before letting you touch one. This module gets you to your first result in under 5 minutes. The theory comes after you've seen it work.

## Lessons

### Lesson 0.1: Your First Terminal
Opens a terminal, explains what it is, and gets you to run `whoami`. The goal: demystify the blinking cursor.

### Lesson 0.2: Installing Qwen Code
Install Qwen Code via npm and launch it. The goal: get the tool running.

### Lesson 0.3: Your First Qwen Command
Create a file through natural language. The goal: see the AI take action on your words.
